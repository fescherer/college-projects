## üöÇ Getting Started üöÇ

To get this project up and running locally on your pc, follow the steps:

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ofelipescherer/college-projects.git
   ```
2. Go the TCC files
   ```sh
   cd tcc
   ```
3. Open folder 'frontend' and run:
   ```
   npm install
   npm start
   ```
